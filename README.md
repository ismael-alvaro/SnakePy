# SnakePy

EN | [PT-BR](#PT-BR)

SnakePy is a 2D Python game built with Pygame, inspired by the classic Snake formula. It combines movement, collision, item collection, and a hazard element represented by the bomb, resulting in a lightweight and engaging project.

## Highlights

- Snake movement with growth after collecting items.
- Three collectible types with different scores.
- A bomb obstacle that adds extra challenge.
- Collision detection with walls and with the snake body.
- Modular codebase designed for maintainability.

## How to run

1. Install Python 3 and Pygame.
2. Open the project in your preferred IDE.
3. Run `main.py`.

## Controls

Use the arrow keys to move the snake:

| Action | Keys |
| --- | --- |
| Move | `↑`, `↓`, `←`, `→` |

## Project structure

- `main.py`: application entry point.
- `snake_game.py`: main loop, rules, and user interface.
- `functions.py`: helper functions for drawing, movement, and game over logic.
- `sprites.py`: collectible item definitions and updates.
- `bomb.py`: bomb logic and random spawning.
- `settings.py`: general configuration, paths, and resource loading.

## Tech stack

- Python
- Pygame
- Python standard library modules such as `random` and `os`

# PT-BR

[EN](#SnakePy) | PT-BR

SnakePy é um jogo 2D desenvolvido em Python com Pygame, inspirado no clássico Snake. O projeto combina movimentação, colisão, coleta de itens e um elemento de risco representado pela bomba, criando uma experiência simples, rápida e divertida.

## Destaques

- Movimento da cobra com aumento de tamanho ao coletar itens.
- Três tipos de coletáveis com pontuação diferente.
- Bomba como obstáculo adicional para aumentar o desafio.
- Colisões com paredes e com o próprio corpo.
- Estrutura modular para facilitar manutenção e evolução.

## Como executar

1. Instale o Python 3 e o Pygame.
2. Abra o projeto em sua IDE preferida.
3. Execute o arquivo `main.py`.

## Controles

Use as setas do teclado para mover a cobra:

| Ação | Teclas |
| --- | --- |
| Mover | `↑`, `↓`, `←`, `→` |

## Estrutura do projeto

- `main.py`: ponto de entrada da aplicação.
- `snake_game.py`: loop principal, regras e interface do jogo.
- `functions.py`: funções auxiliares de desenho, controle e fim de jogo.
- `sprites.py`: definição e atualização dos itens coletáveis.
- `bomb.py`: lógica da bomba e sua geração aleatória.
- `settings.py`: configurações gerais, caminhos e carregamento de recursos.

## Tecnologias

- Python
- Pygame
- Módulos padrão do Python, como `random` e `os`
